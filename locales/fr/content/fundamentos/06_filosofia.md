# Philosophie opérationnelle

## Posture
STP est un environnement de niveau universitaire pour **définir, calculer, comparer, nullifier et documenter** le Tau systémique + RECD — ni tableau de bord générique ni dispositif clinique.

## Honnêteté
- Preuve d’ancrage (CCTP/SDDB) ≠ chaque démo pédagogique.
- Les extensions (TDA, Breathing Window, mémoire ordinale) raffinent le contraste ; elles ne remplacent pas la claim du noyau.
- Methods + hash SHA-256 accompagnent tout résultat sérieux.

## Citation
Citez l’ancre empirique pour les claims cardiaques ; citez les démos comme pédagogiques sinon.
