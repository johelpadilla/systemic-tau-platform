# Limites des EWS classiques

## Objectifs d’apprentissage
- Expliquer pourquoi variance et AR(1) peuvent être silencieuses ou ambiguës près d’une transition.
- Situer le rôle des EWS classiques dans STP : **panneau de comparaison**, pas la claim principale.

## Le récit classique
Les signaux d’alerte précoce (EWS) issus du ralentissement critique (CSD) — hausse de variance, hausse d’AR(1) — sont puissants lorsque le système ralentit vraiment près d’un point de bascule. Beaucoup de séries multivariées réelles (cardiaque, épidémique, écologique) **ne suivent pas** toujours ce chemin de manuel.

## Pourquoi STP les conserve
Les EWS classiques restent dans le Lab pour une lecture duale honnête : le panneau relationnel peut bouger quand le panneau classique ne bouge pas (ou l’inverse). C’est un résultat scientifique, pas un échec d’interface.

## Pratique
Lancez une démo cardio-like ou dengue-like et comparez les courbes de variance/AR1 (z-score) avec τ_s sur les mêmes fenêtres.
