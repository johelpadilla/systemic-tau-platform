# Guide rapide (1 page)

**15–20 minutes jusqu’à la première expérience.**

1. Ouvrez le **Laboratoire**.
2. Chargez les **logistiques couplées** (switch ground truth).
3. Mode **Fast** → Exécuter.
4. Lisez la **lecture duale** : Δτ_s, excess3, p_surr.
5. Exportez Markdown + Methods + hash.

Le mode **Full** active Breathing / TDA / mémoire par défaut (extensions).

Ensuite : Fondements si les concepts sont nouveaux ; Matériels pour les packs imprimables.
