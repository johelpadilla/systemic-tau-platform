# Relación con Teoría de Transiciones Críticas y Critical Slowing Down

## El marco clásico

La teoría de **critical transitions** (Scheffer y otros) establece que sistemas con estados alternativos o bifurcaciones fold pueden mostrar señales genéricas de proximidad al umbral. El mecanismo más citado es el **critical slowing down (CSD)**: el autovalor dominante del jacobiano se acerca a cero y la recuperación se ralentiza.

Eso motiva EWS univariadas (var, AR1, etc.). El marco es poderoso cuando:

- el modelo de potencial es una buena metáfora,
- hay un parámetro de control lento,
- el ruido es aditivo moderado,
- la observación es esencialmente univariada o proyectable a un modo dominante.

## Dónde el paradigma Tau/RECD se alinea

1. **Misma pregunta científica:** ¿podemos anticipar o caracterizar un cambio de régimen?
2. **Misma exigencia de falsabilidad:** nulos, surrogates, controles.
3. **Interés en memoria y correlación:** el CSD predice más memoria; las medidas ordinales también ven cambios de estructura temporal, pero a través de **rangos y conjunciones**.

## Dónde se separa (productivamente)

| Aspecto | CSD clásico | Tau + RECD |
|---------|-------------|------------|
| Mecanismo supuesto | Enlentecimiento de un modo | Reorganización relacional multivariada |
| Firma canónica | var↑, AR1↑ | Δτ_s y Δexcess3 (signo contextual) |
| Tiempo | reloj externo | reloj interno (RECD) |
| Unidad | a menudo 1 serie | N≥2 + niveles Φ |

En datos cardíacos pre-FV, el CSD “de libro” **no** describe bien el AR1 (que suele bajar). Aun así hay transición clínica inequívoca. El marco relacional **sigue viendo señal**. Eso no “refuta” el CSD; muestra que **no es la única vía genérica** en sistemas fisiológicos.

## Síntesis para el usuario avanzado

> Use EWS clásicas como **panel de control univariado**.  
> Use τ_s + RECD como **panel de reorganización relacional**.  
> Una transición robusta es aquella en la que **varios paneles** se mueven de forma coherente *o* en la que el panel relacional explica la discordancia de los clásicos.

La plataforma implementa ambos paneles en el Laboratorio (“Modo comparación con EWS clásicos”) precisamente para entrenar esta lectura dual.
